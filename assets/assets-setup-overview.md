# Assets Overview

This directory contains supporting files for the solution including images, diagrams, sample data files, and other resources referenced throughout the documentation.

**Organization:** Name files descriptively (e.g., `architecture-diagram.png`, `create-ibm-cloud-api-key.png`) and reference them in docs using relative paths like `![Architecture](../assets/architecture-diagram.png)`.

**Tip:** Keep file sizes reasonable (<5MB) and use PNG for diagrams, JPG for photos, SVG for scalable graphics.
