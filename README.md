## Welcome! ##

Use this file to navigate IBM cloud to access the watsonx Orchestrate environment and how to access credentials for the **Agent Development Kit**. Once you are done, you can get proceed to **instructions.md** to begin the lab.  

## IBM Cloud ##

Begin by entering your IBM Cloud credentials.  
![image](https://github.ibm.com/user-attachments/assets/0563b1af-f46b-4794-b8b3-805c105e4c72)

You should now see the following page.  
![image](https://github.ibm.com/user-attachments/assets/28d5062d-b60d-4f86-b6ed-3763d71c1571)  

Click the hamburger icon in the top left, then **Resource List**, and then **watsonx Orchestrate-nsrpoc** under AI/Machine Learning. 
![image](https://github.ibm.com/user-attachments/assets/140b9725-64e1-4fb6-b004-5d1a4ebd948f)

You should see a page like this. 
![image](https://github.ibm.com/user-attachments/assets/9cea1613-5069-41f1-9db3-828680d10556)

Under the **Manage** tab, right click **Access (IAM)**. On the left hand side, click into **API Keys** and create a key with whatever name you'd like. The rest of the settings can be left as is. Copy this key into a notebook/file you can access later since this will be used for accessing the Agent Development Kit.
![image](https://github.ibm.com/user-attachments/assets/696c0cbc-a1d7-4f03-9199-f63fe727f249)

Go back to the previous page and click the blue **Launch watsonx Orchestrate** button.  


In the top right, click on your profile, and right click **Settings**.  
![image](https://github.ibm.com/user-attachments/assets/9b6e9c61-abab-4d24-938f-c49f44f6c38e)

Go to the **API Details** tab and now you should see the endpoint URL being used. Copy this key into a notebook/file you can access later.  
![image](https://github.ibm.com/user-attachments/assets/48c6a398-e978-4cc7-a53c-1884b0b8c678)

Upon finishing these steps, you are ready to begin the lab and should have all credentials needed for the Agent Development Kit.
