# Getting IBM Cloud IAM API Key
### 1. Navigate to IBM Cloud
#### Here: [https://cloud.ibm.com/resources](https://cloud.ibm.com/resources)

<img width="1000" alt="image" src="./assets/create_api_key/ibmcloud-link.png">


### 2. Click on the **Manage** tab and then **Access (IAM)**

<img width="1000" alt="image" src="./assets/create_api_key/ibm-cloud-manage-access.png">

### 3. Click on the **Hamburger** Button and then **API Keys**

<img width="1000" alt="image" src="./assets/create_api_key/ibm-cloud-manage-api-keys.png">


### 4. Click on the **Create** Button

<img width="1000" alt="image" src="./assets/create_api_key/ibm-cloud-api-keys-create.png">

### 5. Give it a Name (I usually name it based on the instance and or project), and then click the **Create** Button

<img width="1000" alt="image" src="./assets/create_api_key/ibm-cloud-api-keys-name.png">

### 6. **Copy** the API Key and **Download** to keep track of, you will be putting this into your [.env](./.env)

<img width="1000" alt="image" src="./assets/create_api_key/ibm-cloud-api-key-copy.png">
